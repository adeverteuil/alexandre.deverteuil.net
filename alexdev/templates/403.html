Français

Erreur 403 : Accès refusé.

« Le serveur a compris la requête, mais refuse de l'exécuter. [⋯] Cela signifie généralement que l'authentification a été acceptée mais que les droits d'accès ne [vous] permettent pas d'accéder à la ressource. »
— Wikipédia. Liste des codes HTTP, Erreur du client.

English

Error 403: Forbidden.

“[This error code] indicate that the server can be reached and understood the request, but refuses to take any further action.”
— Wikipedia. HTTP 403.
“The request was a valid request, but the server is refusing to respond to it. Unlike a 401 Unauthorized response, authenticating will make no difference.”
— Wikipedia. List of HTTP status codes, 4xx Client Error.
